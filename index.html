PHP之文件操作

PHP之文件操作

<form enctype="multipart/form-data">

这个表单属性是使用文件上传所需的一种特殊型编码

默认上传到临时文件夹中

$_FILES

表单通过$_FILES全局变量将有关文件的一些有用信息传递到PHP脚本

$_FILES['screenshot']['name'] 上传文件的文件名
$_FILES['screenshot']['type'] 上传文件的MIME类型
$_FILES['screenshot']['size'] 上传文件的大小
$_FILES['screenshot']['tmp_name'] 文件在服务器上的临时存储位置
$_FILES['screenshot']['error'] 文件上传的错误代码，0表示成功，其他表示失败

move_uploaded_file

move_uploaded_file()函数接收一个文件的源位置和目标位置，然后完成对文件的移动

示例：move_uploaded_file($_FILES['screenshot']['tmp_name'], $target);

is_file

使用is_file()函数判断给定文件是否是一个正常文件

filesize()

使用 filesize()函数取得文件大小

注意

is_file 及 filesize 函数进行文件检查的时候需要文件的完整路径，如果只传文件名is_file将会返回false，filesize函数则会返回
Warning: filesize(): stat failed for img.jpg

设置mysql字符集

设置mysql字符集

用php插入中文数据的时候一直插入失败，检查后发现是mysql的字符集设置有问题

解决方法

修改 /etc/my.cnf 文件
在 [mysqld]中添加
character-set-server=utf8
在 [client]中添加
default-character-set=utf8
在 [mysql]中添加
no-auto-rehash
default-character-set=utf8

注意

以上方法只对新创建的数据库起作用，如果需要更改当前数据库使用的字符集
方法：ALTER DATABASE [数据库名] CHARACTER SET utf8; 输入命令的时候请去掉中括号
